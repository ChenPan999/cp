# My CP Project  
![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)  

## 简介  
基于 PubMed 的医学知识图谱问答系统，支持自然语言查询和可视化展示。  

## 功能特性  
- 从 PubMed 自动抽取实体关系  
- 知识图谱可视化  
- Cypher 查询生成  

## 快速开始  
```bash
git clone https://github.com/ChenPan999/cp.git
cd cp
pip install -r requirements.txt
streamlit run app.py
